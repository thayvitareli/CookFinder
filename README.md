## 🍽️ Cookie Finder

Um aplicativo para listar receitas utilizando a API TheMealDB.

🚀 Funcionalidades iniciais

- Listar receitas

- Visualizar detalhes de uma receita

### 🛠️ Tecnologias Utilizadas

- React Native (com Expo)

- Axios (para chamadas de API)
